# Orino Smart Village

OrinoAr rebuilt using Flutter (Dart) language.

## Installation

Follow this guide to install Flutter on your computer: [Link](https://docs.flutter.dev/get-started/install)

## Features
- Onboarding screen
- QR code scanner
- Map with locations
- Fetch post using Wordpress API
